Place the shortcut file "Enable_DPI.bat" on the Desktop or other convenient place of your choice.
The shortcut points to "Enable_DPI.bat" Windows Batch File which points to "Enable_DPI.ps1" Windows PowerShell Script file.
The files "Enable_DPI.ps1" and "Enable_DPI.bat" must be in the same folder.
Use the script to set the value to "2" to have the resolution setting of "Let the operating system scale the resolution" in Citrix.
